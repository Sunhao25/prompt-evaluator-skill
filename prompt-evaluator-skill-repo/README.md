# Prompt Evaluator (Agent Skill)

This repository contains an **Agent Skill** named **prompt-evaluator**.

## Install (any supported agent)

```bash
npx skills add <owner>/<repo>
```

## List available skills in this repo

```bash
npx skills add <owner>/<repo> --list
```

## Install only this skill (optional)

```bash
npx skills add <owner>/<repo> --skill prompt-evaluator
```

## Notes

- Skills are discovered from folders containing a `SKILL.md` file.
- To appear on skills.sh, your repo must be public and someone must install it using the CLI (telemetry must not be disabled).
